1. Проектирование веб-ориентированной системы для подготовки к ЕГЭ

Цель исследования: Разработка архитектуры и функциональной модели веб-приложения для автоматизации процесса подготовки к государственному экзамену.

Задачи исследования: Проанализировать требования к КИМ ЕГЭ по информатике; спроектировать структуру базы данных; реализовать интерфейсы для ученика и администратора.

Эксперимент / методология: Использование объектно-ориентированного анализа и CASE-средств (диаграммы UML) для проектирования логики взаимодействия пользователя с контентом.

Достигнутые результаты: Создан прототип информационной системы, включающий модули тестирования, теоретический блок и систему мониторинга прогресса.

Практическая значимость: Готовая модель данных может быть использована как база для разработки любого образовательного портала со сложной структурой заданий.
